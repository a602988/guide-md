#  權限角色管理

您可在此新增後端管理員的角色權限，與會員等級 。

### ✔ 新增管理群組角色


1. 於列表中，點選右上角按鈕 `新增` 
2. 進去新增頁面，設定以下內容 ( 範例01 )，
3. 下方按鈕 `送出`
4. 於列表中，依據剛建立的那筆資料，點選按鈕(鉛筆) `編輯`
5. 設定 *權限設定*
3. 下方按鈕 `送出`
